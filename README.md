# Vitamin Deficiency Prediction System

This project predicts possible vitamin deficiencies using machine learning techniques based on health and lifestyle factors.

## Technologies Used
- Python
- Machine Learning
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Project Description
The system analyzes user health data and predicts vitamin deficiency patterns using machine learning algorithms. The model was evaluated using a confusion matrix and achieved an accuracy of 63%.

## How to Run
1. Install required libraries
2. Open the Jupyter Notebook file
3. Run all cells to see the prediction results
